ISSI Test Tools - Getting Started Guide

The ISSI Test Tools (ITT) is a software tool designed for the conformance and performance testing of the ISSI specifications TIA-102.BACA and TIA-102.CACC.



Software Download

The ITT is version controlled under subversion on the java.net site.

P25-wireline Home

Two favorites of download are available. The itt-install jar file will install the tool on user's computer on a single click. The issi-tools zip file contains the source code for tool developers. The ITT had been tested under the Window-XP and Linux operating system.



Software Documentation

The software documentations can be found under [install_dir]/doc/index.html.



Prerequisites

The ITT is developed in java programming language. Before you can launch the tool, you will need to install java JRE 1.6 or JDK 1.6 and the jpcap runtime library.

java JDK or JRE download

jpcap runtime library



ITT Software Components

After installation, there will be 3 icons available on desktop to launch the ITT software components:

  1. Conformance Console
  2. Standalone Emulator
  3. Performance Trace Analyzer

The Conformance Console and Standalone Emulator are used to perform ISS conformance testing in distributed and standalone mode respectively. The Standalone Emulator is very useful during the development of the ITT software. The Conformance Console is used for the actual testing of RFSSs in distributed environment.

The Performance Trace Analyzer is used to analyzer the performance of RFSSs from the pcap file captured using the WireShark. It calculates the performance parameters from the captured traces in accordance to ISSI specifications.



ITT Users

The ITT user will use one of icons to launch the desired ITT components.



ITT Developers

Before any development, ITT developer will be required to install ant 1.7 or later.
ant download

After unzipping of the source code, ant tool can be used to build the ITT:
> ant clean
> ant compile
> ant installer

To launch ITT Console in a standalone mode:
> ant emulatorGUI

To launch ITT Performance Analyzer:
> ant analyzerGUI
