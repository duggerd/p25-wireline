ITT - Conformance Tester

Introduction

The ISSI Tester tool is designed to be a distributed test system which will enable users to run conformance tests in accordance to the ISSI Specification (TIA-102.BACA).



User Interface Layout

The user interface area is divided into the top(Conformance Tests) and bottom(Test Traces) panels. These panels are further subdivided into left and right panels with context information.


The Top- Conformanace Tests panel consists of a list of conformance tests on the left and the test description on the right panel. Depending on the test being selected, the corresponding test details will be displayed on the right-hand-side tabbed panes.


The Bottom- Test Traces panel consists of test traces(SIP and PTT) of each test runs on the left panel and the corresponding message details on the right panel.

Test Procedures

In general, the ISSI Conformance Test procedures are as follows:

  • Test Setup
  • Generate Trace
  • Run Test

These options are available on the top left panel under the conformance tests lists panel.


Test Setup

When the ISSI Tester first launched using the ITT Console or from the application pulldown menu, the current host IP address will be displayed on the title bar. At this time, the only option allowed will be "Test Setup" for the selected conformance testcase.


The Test Setup dialog allows user to configure the IP addresses of the RFSSes involved in the test. The menu also allows uses to change the name, id, port, and other parameters. Once confirmed, the tester will setup test configuration and distribute the topology to the RFSSes involved.

After the Test Setup is performed, the other options will be available to user.


Generate Trace

The Generate Trace option is used to generate reference traces of the selected conformance test. The Generate Trace must be performed before the test can be run. The reference traces will be used to grade the test traces for pass and fail.



Run Test

After the reference traces are generated, the conformance test can be run. The SIP and PTT traces will be captured and compared with the reference traces in grading the test.



Test Results

The test description and test results conatined in the top and bottom panels can be saved on demand for future references.



Log Files

The ISSI Tester maintains various run-time log files under the installed directory /logs. The most useful being the debuglog.txt and guidebuglog.txt files just to name a few. The motivated users are encouraged to consult these log files when a problem is encountered.


Test Profile Setup

The Conformance Tester is pre-loaded with all test cases defined in the TIA-102.CACC specificaition. The Test Profile Setup option under the Test pulldown menu can be used to define the user's test profile from the existing list or it's custom test.
