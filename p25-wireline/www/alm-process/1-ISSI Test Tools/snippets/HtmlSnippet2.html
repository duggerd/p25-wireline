This version contains fixes for close to 85% of bug reports contained in the issue tracker database.

Note:  After downloading the binary file on a machine running Windows XP, the Windows machine may recognize the file as a .zip rather than a .jar.  For java to execute the binary the file must have a .jar extention.  The file must be renamed through the DOS command line.  The command that needs to be issued will be similar to the following: 

"rename itt-install-1.0.rc3.zip itt-install-1.0.rc3.jar"

 The file name prefix will vary depending on the release of the software you are using.
